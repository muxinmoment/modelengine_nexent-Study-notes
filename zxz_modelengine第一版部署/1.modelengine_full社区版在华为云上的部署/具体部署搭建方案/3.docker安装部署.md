docker在华为云的安装部署步骤如下：

1.安装依赖包（让系统支持 HTTPS 源）
sudo apt update && sudo apt install -y \
  apt-transport-https \
  ca-certificates \
  curl \
  software-properties-common \
  gnupg-agent

2.添加 Docker 官方 GPG 密钥（验证软件包安全性）
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/docker.gpg

3.添加官方 Docker 源（外网直连）
# 确定Ubuntu版本代号（如jammy、focal等）
UBUNTU_CODENAME=$(lsb_release -cs)

# 添加官方ARM64源（关键：明确指定arm64架构）
sudo add-apt-repository "deb [arch=arm64] https://download.docker.com/linux/ubuntu $UBUNTU_CODENAME stable"

3.1.添加华为云 ARM 专属 Docker 源(ps：如果像我一样选的香港节点不用设置镜像，直接使用官方docker源就行了)
# 固定架构为 arm64，系统版本代号通过 lsb_release -cs 获取（如 22.04 是 jammy）
sudo add-apt-repository "deb [arch=arm64] https://mirrors.huaweicloud.com/docker-ce/linux/ubuntu $(lsb_release -cs) stable"

4.安装 ARM 版本 Docker 引擎
# 更新源缓存（确保识别到 ARM64 包）
sudo apt update
# 安装 Docker CE（社区版，ARM64 专属版本会自动匹配）
sudo apt install -y docker-ce docker-ce-cli containerd.io

5.启动 Docker 并设置开机自启
sudo systemctl start docker
sudo systemctl enable docker
# 查看状态（active(running) 即为正常）
sudo systemctl status docker

如果是国内服务器可能还需要配置镜像源，我是香港的服务器就不配置了
